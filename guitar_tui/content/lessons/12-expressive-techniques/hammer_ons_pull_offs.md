## What Are They?

A **hammer-on** sounds a higher note by pressing a finger firmly onto the string — without picking. A **pull-off** sounds a lower note by snapping the fretting finger off the string sideways.

Together these techniques produce **legato** phrasing: smooth, connected phrases where only some notes are picked. In tab:

- `h` between two fret numbers means hammer on to the second
- `p` between two fret numbers means pull off to the second

## Hammer-on

Fret the first note and pick it. While the string is still ringing, press your next finger onto the higher fret firmly:


Pick only the first note. The second sounds from the hammer alone. Both notes must ring at equal volume — a weak hammer produces a ghost note that gets buried in the mix.

## Pull-off

Fret both notes simultaneously, pick the higher one, then snap the fretting finger off the string sideways:

```diagram
type: tab
title: Pull-off — B string
lines:
  - beats:
      - notes: [null, null, null, null, 8, null]
        label: "pick"
      - notes: [null, null, null, null, 5, null]
        technique: "p"
```

The snap must go **sideways** across the string, not straight off the fretboard. The sideways motion plucks the string and generates the lower note. Lifting straight off produces no sound.

## A Legato Run

Combining hammer-ons and pull-offs lets you produce many notes from few picks:

```diagram
        label: "pick"
      - notes: [null, null, null, null, null, 8]
        technique: "h"
      - notes: [null, null, null, null, null, 5]
        technique: "p"
```

Two picks, five notes. The hammers and pull-off fill in the remaining three for free, creating a fluid, connected sound that alternate picking cannot match at speed.

## Technique Points

**Hammer-on**: land the fingertip close to the fret wire, not in the middle of the fret. A landing near the wire requires far less force to produce a clear note and reduces the risk of buzzing.

**Pull-off**: the lower finger must already be in position on its fret before you pull off. Set up the shape first, then snap. If the lower finger is not planted, there is nothing for the string to ring against.

**Volume balance**: hammered and pulled notes tend to be quieter than picked notes. Work toward matching volumes before increasing tempo.

## Practice

Isolate each technique as a two-note motion before combining them. One clean, even hammer-on is worth more than ten sloppy ones. Once both techniques sound equal in volume to a picked note, string them into the full run above.
