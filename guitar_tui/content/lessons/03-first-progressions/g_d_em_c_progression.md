## Why This Progression

G, D, Em, and C are the four most-used chords in the *G major scale*. They appear together in more songs than almost any other combination — folk, pop, rock, and country all rely on them constantly.

Learning these four shapes is one thing. Moving between them without a pause is the real skill, and that is what this lesson builds toward.

## The Four Chords

### G Major

You have already met this chord. Three fingers, all six strings ring freely.

```diagram
type: chord
title: G Major
frets: [3, 2, 0, 0, 0, 3]
fingers: [2, 1, null, null, null, 3]
```

Keep your **ring finger** well arched on the high `e` string — a flat fingertip will mute the B string next to it.

### D Major

D uses only the top four strings. The low `E` and `A` strings must be muted — rest the fleshy base of your strumming hand lightly against them, or simply avoid them with your pick.

```diagram
type: chord
title: D Major
frets: [null, null, 0, 2, 3, 2]
fingers: [null, null, null, 1, 3, 2]
```

- **Index finger** — 2nd fret, G string
- **Ring finger** — 3rd fret, B string
- **Middle finger** — 2nd fret, high e string

The three fingers form a tight triangle. Keep them bunched close to the frets so each note rings clearly.

### E Minor

Em is one of the most forgiving chords on the guitar — only two fingers, and four strings ring open. It also sits naturally close to G major, making the G → D → Em move very fluid once you have the shapes.

```diagram
type: chord
title: E Minor
frets: [0, 2, 2, 0, 0, 0]
fingers: [null, 2, 3, null, null, null]
```

All six strings ring. Strum slowly and listen for the open `G`, `B`, and `e` strings — they give Em its characteristic open, resonant sound.

### C Major

C is the most demanding chord in this group. The index finger must arch steeply at the first knuckle so the open `B` string directly below it can ring freely.

```diagram
type: chord
title: C Major
frets: [null, 3, 2, 0, 1, 0]
fingers: [null, 3, 2, null, 1, null]
```

- **Index finger** — 1st fret, B string (tip must not touch high e)
- **Middle finger** — 2nd fret, D string
- **Ring finger** — 3rd fret, A string
- Mute the low `E` string — do not strum it

Pluck each string one at a time after placing C to confirm that the `G` and `e` strings ring open. If either is muted, adjust your index finger arch.

## Practising the Transitions

Do not practise the full loop until each individual change feels comfortable. Work on pairs:

1. **G → D** — the hardest. All three fingers relocate completely. Practise this pair most.
2. **D → Em** — the easiest. A good pair to warm up with.
3. **Em → C** — moderate. The index finger drops onto the B string while the other two shift up.
4. **C → G** — satisfying once it clicks. The ring finger anchors on the A string as the others spread.

**One-minute changes** — set a timer for 60 seconds and alternate between just two chords, strumming once per second. Count only the changes where both chords sound clean. Aim for 30 before moving to the next pair.

## Common Problems

- **Buzzing strings** — finger is not close enough to the fret wire, or is touching an adjacent string. Move the fingertip closer to the wire.
- **Muted open strings** — a neighbouring finger is leaning into an open string. Arch more at the knuckle.
- **Slow transitions** — normal at first. Use one-minute changes daily. Speed is a byproduct of repetition, not effort.
- **Tension in the hand** — squeeze only as hard as needed to stop the buzz. Excess grip causes fatigue and slows transitions.

## Next Steps

Once you can loop G–D–Em–C cleanly at a comfortable tempo, try reversing the order (C–Em–D–G) — it has a different emotional feel and exercises the same transitions in the opposite direction.

The [lesson:minor_pentatonic_intro] scale sits perfectly over this progression when played in the key of `E` minor. Once both are comfortable, you have everything needed to improvise your first solo.
