## The Exercise

Set a timer for 60 seconds. Alternate between two chords as many times as
possible. Count only *clean* changes — both chords must ring fully.

### Recommended Pairs (in order of difficulty)

| Pair | Why it is useful |
|------|-----------------|
| G → D | Share the 3rd finger anchor; teaches pivot |
| D → Em | Large shift; one of the hardest beginner transitions |
| C → G | Very common; C to G requires re-anchoring all fingers |
| Am → E | Common in minor-key songs; develops ring finger independence |
| G → Em | Easy pivot; good starting point for beginners |

## How to Run the Exercise

1. Pick one pair. Set a timer for 60 seconds.
2. Strum one downstroke per chord. Switch. Repeat.
3. Count every full clean cycle (both chords = 1 count).
4. Record your score. Rest 60 seconds. Repeat 3–5 times.
5. Next session, aim to beat your best score.

## Rules

- No partial changes count. If the chord buzzes, it does not count.
- Do not strum until both hands are fully in position.
- Speed comes from *efficient movement*, not rushing. Watch your fretting hand:
  are all fingers moving simultaneously, or one at a time?

## Target Scores

| Score | Level |
|-------|-------|
| Under 20 | Needs work — focus on form |
| 20–35 | Progressing — keep the daily habit |
| 35–50 | Solid — ready for strumming patterns |
| 50+ | Strong — move to harder pairs |
