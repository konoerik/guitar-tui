## The Exercise

Play an E-shape barre chord at fret 5 (A major), release completely, then play
an A-shape barre chord at fret 5 (D major). Both chords share the root note A
on the low E string at fret 5. Release fully between each chord — the hand
must reform the shape from scratch each time, not just relax and squeeze.

```diagram
type: chord
title: E-Shape at fret 5 — A Major
frets: [5, 7, 7, 6, 5, 5]
fingers: [1, 3, 4, 2, 1, 1]
barre: {from: 1, to: 6, fret: 5}
```

```diagram
type: chord
title: A-Shape at fret 5 — D Major
frets: [null, 5, 7, 7, 7, 5]
fingers: [null, 1, 3, 3, 3, 1]
barre: {from: 2, to: 6, fret: 5}
```

Repeat: E-shape → release → A-shape → release. Count clean changes for
60 seconds (same scoring as One-Minute Changes).

## Rules

- **Both chords must ring fully** — arpeggiate each chord before releasing to
  confirm every string sounds. Do not count a change if any string buzzes.
- Complete hand release between shapes. No partial holds.
- Start at a single position (fret 5) before moving up and down the neck.
- Once stable at fret 5, repeat at fret 7 (B major / E major), then at fret
  3 (G major / C major).

## Why This Exercise

Most barre chord difficulty is not strength — it is shape inconsistency.
Players who struggle with barre chords usually have one finger not fully
flattened, or a thumb position that limits reach. By isolating the two
shapes and releasing fully between each change, this exercise forces you
to rebuild the shape correctly every repetition. The problem spot (if any)
will become obvious: one chord will consistently buzz on the same string.

## Common Mistakes

**Thumb too high on the neck.** The thumb should sit behind the middle finger,
roughly centered behind the barre. If it creeps above the neck, the barre
finger cannot flatten properly.

**Only partially releasing between shapes.** If the hand does not release
fully, muscle tension carries over and the next shape forms incorrectly. Take
a full breath between each shape at first.

**Barring all six strings when only five are needed.** The A-shape barre only
requires strings 2–6; string 1 (high e) is either muted or open. A lighter
touch on string 1 reduces hand fatigue without sacrificing the chord sound.
