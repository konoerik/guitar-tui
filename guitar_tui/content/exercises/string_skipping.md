## The Exercise

Most scale and arpeggio patterns move between adjacent strings. String skipping jumps over a string with every pick stroke, which demands a different kind of picking-hand precision. The right hand must travel further and land accurately without grazing the skipped string.

This exercise runs the A minor pentatonic position 1 as a string-skip sequence. Each beat pairs a note on one string with a note on the string two positions away — skipping the string in between.

The pattern pairs strings 6+4, then 5+3, then 4+2, then 3+1 — each pair skips the string between them. The ascending note pairs are: A/G, D/C, A/E, C/A.

## Rules

- Start at **60 BPM**, eighth notes. Do not increase tempo until every note rings cleanly with no inadvertent muting of the skipped string.
- Use **strict alternate picking** throughout. The right hand must swing past the skipped string without touching it — this is the core challenge.
- Do not tilt the pick to avoid the skipped string. Instead, control the arc of the picking motion. The angle stays consistent; the travel distance increases.
- Once comfortable ascending, reverse and descend (high e → G string, working back to strings 6+4).

## Common Problems

- **Muting the skipped string accidentally**: usually caused by flattening the pick trajectory. Keep the pick perpendicular to the strings and let the motion arc cleanly between the target strings.
- **Uneven volume**: the note on the far string (the second note of each pair) is easier to hit with less force. Consciously match the volume of both notes in each pair before increasing tempo.

## Progression

1. Master the ascending pattern (this exercise)
2. Descend in the same pairs, reversed
3. Combine: ascend through all four pairs, then descend immediately
4. Apply the same skip pattern to natural minor and major scale positions — the technique is scale-agnostic, but changing the note content keeps the ear engaged
5. Try two-string skips (strings 6+3, 5+2, 4+1): harder, but the same principle
