## The Exercise

The five CAGED-based major scale positions use two or three notes per string
inconsistently. The three-notes-per-string (3NPS) pattern uses exactly three
notes on every string — producing a consistent pick pattern (down-up-down or
up-down-up on every string) and enabling efficient position shifts.

## Rules

- **60 BPM**, sixteenth notes (three notes = one beat of a triplet group).
  Alternatively: **80 BPM**, eighth-note triplets.
- Strict alternate picking. The three-note grouping means each string starts
  on the opposite pick direction from the previous string — this is intentional
  and builds bidirectional string crossing.
- Keep the fretting hand fingers close to the fretboard between notes.
- On the position shift (when moving from one string group to the next), do
  not pause. The shift happens during the pick motion, not after it.

## Why this pattern matters

The CAGED positions are excellent for understanding fretboard geography. The
3NPS pattern is what fast melodic playing actually uses — the consistent
pick grouping means muscle memory transfers from one key to any other key
using the same fingering logic. Every major scale run in rock and blues
soloing that covers more than one octave is built on this pattern or a
close variant.

## Progression

1. Learn the full ascending + descending pattern in one key (C major, as above)
2. Shift the same fingering to G major (one position lower) and A major
3. Apply sequences: play the pattern in groups of three (1-2-3, 2-3-4, 3-4-5)
4. Apply to natural minor using the same 3NPS logic from the relative minor root
