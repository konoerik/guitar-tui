## The Exercise

Playing the major scale in thirds means alternating between a note and the note a diatonic third above it: A then C#, B then D, C# then E, and so on. The resulting melody — A C# B D C# E D F# — has a natural, song-like quality because diatonic thirds are the basic building block of chord harmony. Every note pair implies a chord: A+C# implies A major, B+D implies Bm, C#+E implies C#m.

The pattern alternates between string 6 (low E) and string 5 (A): play the lower note on string 6, then its third on string 5, then step up one scale degree on string 6, and so on. After the pattern reaches the top of the string pair, continue on strings 5+4, then 4+3, and so on up the neck.

**A major: A B C# D E F# G#. The thirds above each degree: A→C# (major), B→D (minor), C#→E (minor), D→F# (major), E→G# (major), F#→A (minor), G#→B (minor). Diatonic thirds alternate between major and minor — that pattern is a fundamental property of the major scale.**

## Rules

- Start at **65 BPM**, one note per beat. Keep the tempo slow enough to hear each note pair as a musical interval, not just a mechanical exercise.
- Use **alternate picking** throughout. The string-crossing motion (string 6 → string 5 → string 6) trains the same coordination as the string-skipping exercise.
- Say the note names as you play them: "A... C-sharp... B... D..." This builds the ear-to-fretboard connection that makes the exercise musical rather than gymnastic.

## Extending the Pattern

Once the first octave feels natural:

1. Continue on strings 5+4, then 4+3, and so on up the neck
2. Descend in thirds (reverse the direction: F# down to D, E down to C#, etc.)
3. Combine ascending and descending: go up two scale degrees in thirds, then reverse — this creates longer melodic arcs
4. Apply the same pattern to natural minor: the thirds are different (minor scale has b3, b6, b7) and the sequence of major/minor thirds shifts, but the technique is identical

## Why This Works

Running scales in thirds forces you to hear the scale as harmony, not just a sequence of steps. After practicing this exercise, scale runs that jump over a note will start to sound intentional — because they are: you are outlining chord tones. This is the technique behind nearly every lyrical guitar melody that doesn't sound like a "scale run."
